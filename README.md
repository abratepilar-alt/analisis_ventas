# Analisis de Ventas

## Descripcion del Proyecto

Este proyecto fue desarrollado como trabajo practico utilizando Git, GitHub y Google Colab.

El objetivo es aplicar buenas practicas de control de versiones, introducir conceptos fundamentales de Git y GitHub, aplicar comandos fundamentales, trabajo colaborativo y organizacion profesional de repositorios.
